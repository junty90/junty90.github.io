JQuery 使用指南

2020-01-26

文章目录

使用前置： 首先引用 jQuery 文件， 创建 jQuery对象使用 $() 函数，CSS选择器作为参数。

  1. jQuery 如何获取元素

    1
2
3
4

     $(document) //选择整个文档对象
 $('#myId') //选择文档中ID为myId的一个元素
 $('div.myClass') // 选择文档中类为myClass的所有div元素     
 $('img[alt]') // 选择带有alt属性的所有图像元素

  2. jQuery 如何移动元素

    1
2
3
4
5

     $('div').next('p'); //选择div元素后面的第一个p元素
 $('div').parent(); //选择div元素的父元素
 $('div').closest('form'); //选择离div最近的那个form父元素
 $('div').children(); //选择div的所有子元素
 $('div').siblings(); //选择div的同级元素

  3. jQuery 的链式操作是怎样的

    1
2
3
4
5

    // 多个方法连接使用
 $('div#app') //选择id为app的div元素
    .parent() //取得它的父元素
    .addClass('highlight') //为该单元格添加hightlight类
    .end() //恢复到id为app的div元素的父元素
