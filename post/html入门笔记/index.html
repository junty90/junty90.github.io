HTML 章节标签

  • h1~h6 - 表示标题
  • section - 表示章节
  • article - 表示文章
  • p - 表示段落
  • header - 表示头部
  • footer - 表示页脚
  • main - 表示主要内容
  • aside - 表示无关内容
  • div - 表示划分容器

HTML 内容标签

  • ul + li - 表示无序列表
  • ol + li - 表示有序列表
  • dl + dt + dd - 表示描述列表
  • pre - 表示预定义格式文本
  • hr - 表示段落元素之间的主题转换
  • br - 表示换行（回车）符号
  • a - 表示超链接
  • em - 表示强调
  • strong - 表示重要
  • code - 表示等宽字体显示
  • q - 表示短引用
  • blockquote - 表示长引用

标签全局属性

  • id - 表示定义唯一标识符
  • class - 表示类名
  • style - 表示CSS样式声明
  • title - 表示元素相关信息
  • contenteditable - 表示元素是否可被用户编辑
  • tabindex - 表示元素是否可以获取输入焦点（可聚焦），是否应该参与顺序键盘导航，如果是，则表示哪个位置
  • hidden - 表示隐藏
