URL

参考:

https://zh.wikipedia.org/wiki/%E7%BB%9F%E4%B8%80%E8%B5%84%E6%BA%90%E5%AE%9A%E4%BD%8D%E7%AC%A6

URL 全称统一资源定位符（Uniform Resource Locator）

URL包含以下部分：

  • 协议
  • 域名
  • 端口
  • 路径
  • 查询字符
  • 锚点
1
2
3

例子：
https://developer.mozilla.org/zh-CN/search?q=CSS
https://developer.mozilla.org/zh-CN/docs/Glossary/CSS#%E6%8A%80%E6%9C%AF%E8%B5%84%E6%96%99
  • https 指协议
  • developer.mozilla.org 指域名
  • 443 是https的端口（默认不显示）
  • /zh-CN/search 表示路径
  • ?q=CSS 表示以「?」字符为起点，每个参数以「&」分隔，再以「=」分隔参数名和查询名
  • #%E6%8A%80%E6%9C%AF%E8%B5%84%E6%96%99 表示以「#」字符为起点定位页面标题

DNS

参考:

https://zh.wikipedia.org/wiki/%E5%9F%9F%E5%90%8D%E7%B3%BB%E7%BB%9F

DNS 全称网域名称系统（Domain Name System）是互联网的一项服务。

它的作用是将域名和IP地址相互对应。

IP

参考:

https://zh.wikipedia.org/wiki/IP%E5%9C%B0%E5%9D%80

IP 通常指IP地址，全称互联网协议地址（Internet Protocol Address）是分配给网络上使用互联网协议设备的数字标签。

它的作用是定位设备来通信。

域名

参考:

https://zh.wikipedia.org/wiki/%E5%9F%9F%E5%90%8D

网域名称（Domain Name，简称：Domain）是IP地址相应的代称，目的是方便人类记忆。

域名层级：

  • 顶级域名（是域名中最高的一级，每个域名都以顶级域名结尾）
  • 子域名
    • 二级域名（位于顶级域名的左侧）
    • 三级域名（位于二级域名的左侧）
1
2
3
4
5

例如：www.baidu.com

com 是顶级域名
baidu 是二级域名
www 是三级域名
