常用标签学习

  1. a 标签的用法

    作用：

    • 外部页面跳转
    • 内部页面锚点跳转
    • 支持URL、文件、邮箱、电话等链接格式跳转

    常用属性有：

    • href - 指定跳转链接格式
      • 网址
      • 路径
      • 伪协议
      • id
    • target - 指定在哪里显示链接的资源
      • _blank - 新窗口打开
      • _self - 当前页面加载（默认值）
      • _top - 最高层级加载
      • _parent - 父框架加载
      • 自定义名字 - 自定义名字页面打开或加载

  2. img 标签的用法

    作用：发出get请求，展示一张图片

    常用属性有：

    • alt - 图像的替换文本
    • height - 图像的高度
    • width - 图像的宽度
    • src - 图像的地址

    相关JS事件：

    • onload
    • onerror

  3. table 标签的用法

    作用：制作表格

    相关标签：

    • thead
    • tbody
    • tfoot
    • tr
    • td
    • th

    相关样式：

    • table-layout
    • border-collapse
    • border-spacing
