盒模型

在CSS的世界里，每个元素都会渲染成矩形“盒子”，每个盒子由 margin, border, padding, and content ，这4个部分组成，称为盒模型。

引用MDN盒模型的图片：

  • Content box: 这个区域是用来显示内容，大小可以通过设置 width 和 height.
  • Padding box: 包围在内容区域外部的空白区域； 大小通过 padding 相关属性设置。
  • Border box: 边框盒包裹内容和内边距。大小通过 border 相关属性设置。
  • Margin box: 这是最外面的区域，是盒子和其他元素之间的空白区域。大小通过 margin 相关属性设置。

在CSS中的box-sizing属性里定义盒模型的类型有内容盒子 content-box（默认值）、边框盒子 border-box 应用于块级盒子。

尺寸计算公式：

​ 内容盒子的 width = 内容的宽度 ， height = 内容的高度

​ 边框盒子的 width = border + padding + 内容的宽度 ，height = border + padding + 内容的高度
